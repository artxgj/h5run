h5Run is a web application that tracks runs using GPS. It stores GPS data of runs in the browser's database. The data can be viewed even if the phone is not connected to the internet. These two features are made possible by the use of HTML5's browser database storage and application cache. h5Run was originally written for iPhone Safari but has been tested on an Android 2.2 phone.
